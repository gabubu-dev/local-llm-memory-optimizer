# Local LLM Memory Optimizer

A tool for optimizing memory usage of local Large Language Models.

## Overview

This project provides utilities to analyze and optimize memory consumption when running LLMs locally.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/gabubu-dev/local-llm-memory-optimizer.git
cd local-llm-memory-optimizer

# Install dependencies
pip install -r requirements.txt
```
